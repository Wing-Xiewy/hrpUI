检索框需要使用 jQuery 方法来实现匹配和渲染功能：

const list = [
   { id: '001', searchName: '测试名称1', searchValue: 'S1' },
   { id: '002', searchName: '测试名称2', searchValue: 'S1' },
   { id: '003', searchName: '谢婉玉测试', searchValue: 'XWY' }
]
$('#autoDemo').focus(function(e){
   <!-- 检索框获取焦点的时候，匹配和渲染下拉列表 -->
   searchQueryAuto($(this),list,'searchName','searchValue',true);
})
$('#autoDemo').bind('input propertychange', function() {
   <!-- 检索框输入的时候，匹配和渲染下拉列表 -->
   searchQueryAuto($(this),list,'searchName','searchValue',true);
})

searchQueryAuto($(this),List,Name,Value,Bool)方法用来匹配和渲染检索列表：
List 代表需要进行检索的目标列表，类型为 array；
Name 代表List中表示显示值的键，类型为 string；
Value 代表List中表示真实值的键，类型为 string；
Bool 代表检索类型是否启用前台检索，类型为 boolean，如果启用前台检索，则会在前端对输入值进行匹配，并渲染匹配后的List；如果不启用前台检索，则List只会进行渲染，不会自动匹配：

$('#autoDemo').focus(function(e){
   <!-- 检索框获取焦点的时候，渲染下拉列表 -->
   const list = [
      { id: '001', searchName: '测试名称1', searchValue: 'S1' },
      { id: '002', searchName: '测试名称2', searchValue: 'S1' },
      { id: '003', searchName: '谢婉玉测试', searchValue: 'XWY' }
   ]
   searchQueryAuto($(this),list,'searchName','searchValue',false);
})
$('#autoDemo').bind('input propertychange', function() {
   <!-- 检索框输入的时候，渲染下拉列表 -->
   const list = [
      { id: '001', searchName: '测试名称1', searchValue: 'S1' },
   ]
   searchQueryAuto($(this),list,'searchName','searchValue',false);
})

树 Tree

树控件可以完整展现层级关系，并具有展开收起选择等交互功能。

调用方法setTree("#id",[列表],[名称|默认‘name’])可以自动生成树形结构。

调用方法getTreeNode('.hrp-tree-title',key)获取所有末级节点，key为需要获取的键值，如果不设置则默认返回整个对象；该方法可以配合click方法使用，获取选中的末级节点。

<div class="hrp-tree" id="treeExample">
 <!-- 调用 setTree() 方法后，将自动生成下面的树 -->
   <ul class="hrp-tree-children">
      <li class="hrp-tree-subtree">
         <div class="hrp-tree-title">TreeTitle 1</div>
         <ul class="hrp-tree-children">
         ...
         </ul>
      </li>
      <li class="hrp-tree-subtree">
         <div class="hrp-tree-title">TreeTitle 2</div>
      </li>
   </ul>
 <!-- 调用 setTree() 方法后，将自动生成上面的树 -->
</div>

setTree('#treeExample', list)
$('body').on('click', '#treeExample .hrp-tree-title', function(){
   alert(getTreeNode(this,'value'))
})

对于可多选的树，可以使用getTreeCheckedList('#id',key)方法获取选中的节点，key为需要获取的键值，如果不设置则默认返回整个对象。

调用方法setTreeCheckedNode('#id',list,key)设置默认选中的节点，list为选中节点的值，key为节点数据中的一个字段名。

调用方法setTreeOpenedNode('#id',list,key)设置默认展开的节点，list为展开节点的值，key为节点数据中的一个字段名。

<div class="hrp-tree show-checkbox" id="treeCheck"></div>
<button type="button" class="hrp-btn hrp-btn-primary" id="tree-node" onclick="alert(getTreeCheckedList('#treeCheck'))">获取选中的对象</button>
<button type="button" class="hrp-btn hrp-btn-primary" id="tree-key" onclick="alert(getTreeCheckedList('#treeCheck','value'))">获取选中的值</button>

setTreeCheckedNode('#treeCheck',['l111','l112'],'value') setTreeOpenedNode('#treeCheck',['l2'],'value')

遮罩层 Mask

打开模态组件，或需要锁定页面时使用遮罩层。

调用方法openMask(layer,color,type)可以打开自定义遮罩：

layer代表遮罩层的海拔，默认为 500；

color代表遮罩层的颜色，默认为 60% 的黑色；

type代表遮罩层是否可以点击关闭，默认为 false；

<button class="hrp-btn hrp-btn-default" onclick="openMask('','','true')">默认遮罩</button>
<button class="hrp-btn hrp-btn-default" onclick="openMask('2021','rgba(255,255,255,.6)','true')">自定义遮罩</button>

调用方法closeMask()关闭遮罩。
